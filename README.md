# Fraud Detection using Machine Learning

## Overview

This project is a Machine Learning based Fraud Detection system developed to identify suspicious financial transactions from highly imbalanced transaction datasets.
The system uses machine learning algorithms such as Random Forest and XGBoost to classify transactions as fraudulent or non-fraudulent.

---
## Features

* Financial transaction fraud detection
* Data preprocessing and feature scaling
* Random Forest implementation
* XGBoost implementation
* Performance evaluation metrics
* Fraud vs Non-Fraud visualization

---

## Technologies Used
* Python
* Pandas
* NumPy
* Scikit-Learn
* XGBoost
* Matplotlib
* Seaborn
---

## Dataset
The project uses the Credit Card Fraud Detection dataset containing transaction records with fraud labels.
---

## Project Workflow
1. Load dataset
2. Preprocess data
3. Scale features
4. Train models
5. Evaluate performance
6. Visualize results
---

## Models Used
* Random Forest Classifier
* XGBoost Classifier
---

## Results
The models successfully classify fraudulent transactions and provide high accuracy on transaction classification tasks.
